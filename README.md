# fwd [![Build Status](https://secure.travis-ci.org/horse/fwd.png?branch=master)](http://travis-ci.org/horse/fwd)

Simple reverse proxy framework

## Getting Started
Install the module with: `npm install fwd`

```javascript
var fwd = require('fwd');
fwd.awesome(); // "awesome"
```
